# Golf Remote

## Hämta Windows-agenten

Senaste utgåvan: [Golf Remote 0.1.0](https://github.com/lindgren-linus/golf-remote/releases/tag/v0.1.0)

Direkt nedladdning: [GolfRemote-Setup-0.1.0.exe](https://github.com/lindgren-linus/golf-remote/releases/download/v0.1.0/GolfRemote-Setup-0.1.0.exe)

Repo och release är privata. Den som laddar ned behöver därför vara inloggad på GitHub och ha åtkomst till `lindgren-linus/golf-remote`.

## Snabbstart

1. Hämta och kör installationsfilen på simulatorns Windows-dator.
2. Godkänn UAC-dialogen. Installern lägger till de lokala brandväggsregler som behövs.
3. Golf Remote visas som en ikon i Windows systemfält och startar automatiskt vid nästa inloggning.
4. Installera eller öppna mobilappen och anslut till agenten på samma Wi-Fi-nät.

Se [Windows-agent](Windows-Agent) för detaljer och felsökning.
